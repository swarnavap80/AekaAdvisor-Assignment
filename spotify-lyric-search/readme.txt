# Spotify Lyric Search (ML Project)

A machine learning project that identifies the song title and artist
based on a given snippet of lyrics.

## Tech Stack
- Python
- TensorFlow / Keras
- TF-IDF
- Pandas & Scikit-learn

## Features
- Text preprocessing
- Neural network classification
- Accurate lyric-based identification

## Run
python lyric_search_model.py
